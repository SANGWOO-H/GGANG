# GGANG
XAMPP 서버를 활용하여 안드로이드 스튜디오로 GGANG 앱을 만들었습니다.

## 기능
1. 유튜브를 활용하여 깡 뮤직비디오, 깡좌를 시청할 수 있다.
2. 로그인, 회원가입, 로그아웃
3. 자유게시판을 통해 여러 사람들과 소통할 수 있다.
4. 하단메뉴를 통하여 쉽고 간단하게 깡 커버영상을 찍을 수 있다.
